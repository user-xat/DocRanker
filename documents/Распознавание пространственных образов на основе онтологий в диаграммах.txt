Диаграммы широко используются в нашем повседневном общении, и различные типы диаграмм используются для различных целей в разных областях. Известные диаграммы, используемые в вычислительной технике, включают блок-схемы, диаграммы состояний и диаграммы унифицированного языка моделирования (UML). Заметным аспектом этих диаграмм является то, что они имеют общепринятый визуальный синтаксис, который представляет собой пространственные шаблоны, состоящие из графических и текстовых элементов [15, 16] для обозначения специфичных для диаграммы концепций на диаграммах. В этом исследовании рассматриваются только диаграммы, в которых установлен визуальный синтаксис, независимо от того, задан он формально или неформально. Знание этих визуальных шаблонов необходимо для чтения и понимания диаграмм.
Для создания декларативных спецификаций различных аспектов диаграмм, включая их пространственные шаблоны, использовались многочисленные подходы, и эти спецификации использовались для реализации приложений для распознавания или создания диаграмм. Большинство из этих подходов основаны на грамматике, и известность этих подходов обусловлена их успехом в спецификациях естественного языка [15]. Однако количество методов спецификации диаграмм, которые имеют достаточную вспомогательную литературу, свободно доступные инструменты и литературу, указывающую на их текущее использование, остается небольшим.
Онтологии - это известный метод представления знаний, который успешно используется для реализации приложений, основанных на знаниях. Онтология, в общем случае, представляет собой модель знаний предметной области, разработанную для определенной цели. Онтологии используются, как правило, для двух основных целей: как формальный словарь понятий в предметной области или как источник знаний, из которого могут быть получены дополнительные знания. Преимущество онтологий, основанных на логике, заключается в том, что они могут использовать возможности автоматического анализа существующих онтологий для проверки и вывода из них дополнительной информации [5]. Язык веб-онтологий (OWL), язык онтологий на основе логики, имеет богатый набор текущей литературы, доказал свою ценность во многих областях приложений и свободно доступных инструментов [7, 11].
В этой статье мы представляем результаты исследования, в ходе которого был разработан и оценен основанный на онтологиях подход к распознаванию пространственных шаблонов на диаграммах. В этом подходе соответствующие пространственные шаблоны диаграмм кодируются в онтологии OWL, а функция автоматической классификации экземпляров онтологических рассуждений используется для сопоставления пространственных шаблонов с концепциями, специфичными для диаграммы, изображенными на диаграмме. Затем предлагаемый подход реализуется в прототипе для распознавания концепций из файлов масштабируемой векторной графики (SVG) диаграмм классов UML с использованием онтологии соответствующих пространственных шаблонов. Кроме того, производительность прототипа с точки зрения использования времени и корректности проверяется с помощью примерного набора диаграмм классов. Автоматическое распознавание понятий из диаграмм может быть полезным строительным блоком для создания автоматических сводок диаграмм, которые могут, например, быть полезны пользователям с ослабленным зрением при чтении информации, изображенной на диаграммах.
Новый вклад этого исследования состоит из трех частей; во-первых, представлен общий подход, основанный на онтологии, для распознавания концепций, специфичных для диаграмм, из диаграмм. Во-вторых, реализация, хотя и прототип, демонстрирует осуществимость предлагаемого подхода, и, в-третьих, демонстрируется автоматическая семантическая интерпретация ограниченного объема диаграмм классов в формате SVG с использованием онтологии пространственных шаблонов.

Прототип, который следует этому основанному на онтологиях подходу для распознавания концепций UML и предметной области в диаграммах классов формата SVG, представлен в разделе. 3.
